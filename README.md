简介
-------------

休闲连连看是一款免费无广告的经典益智小游戏，清新美观的界面，流畅的操作，炫丽的动画，一定让你爱不释手。
游戏特色：
1.闯关、计时、任务等五种游戏模式
2.超过400个精心设计不同难度的关卡
3.经典，水果，动物，宠物和农场连连看
4.华丽的音效+炫丽的动画
5.网络排行榜，与高手一起比拼
6.对高手点赞+点赞提醒
7.成就一键分享到QQ空间，新浪微博或微信朋友圈
8.搜集钻石和金币，让游戏更有趣味
